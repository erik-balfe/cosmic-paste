# cosmic-paste

Clipboard manager for [COSMIC](https://github.com/pop-os/cosmic-epoch).

**Status:** [`docs/STATE.md`](docs/STATE.md)

## Install

```bash
./scripts/install.sh
```

Then: Settings → Panel → Applets → COSMIC Paste.
